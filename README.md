# Quote Generator
A beautiful, minimalist web application that displays inspirational quotes with the ability to fetch new quotes and share them on Twitter.
Features

* Random Quote Display: Get inspired with random quotes from famous authors and thinkers
* External API Integration: Fetches quotes from the Quotable API for fresh content
* Fallback System: Includes 140+ local quotes as backup if API is unavailable
* Twitter Sharing: Share your favorite quotes directly to Twitter with one click
* Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
* Dynamic Styling: Automatically adjusts font size based on quote length for optimal readability

HTML5: Structure and semantic markup
